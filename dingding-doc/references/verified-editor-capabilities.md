# 已验证的钉钉文档编辑能力

本文件记录已在内置浏览器中验证过的钉钉文档 Web 编辑器非 AI 入口，并给出 agent 可复用的操作方式。未在当前页面验证过的能力不要写成确定路径，先打开对应菜单读取可见文本。

## 已验证入口

在 `alidocs.dingtalk.com` 文档页面中已验证：

- 正文位于 `iframe#wiki-doc-iframe`，可用 `frame.locator('body').innerText()` 读取。
- 工具栏入口包含：`菜单`、`插入`、`默认`、`正文`、`添加图标`、`添加封面`、`设置文档信息`。
- 工具栏入口在渲染文本和截图中可见，但当前页面不一定暴露稳定的 `button` role；自动化前先确认 locator 唯一，点击后必须读取展开菜单或截图确认，不能凭文本可见就假定菜单已打开。
- `插入` 菜单包含：
  - `基础 / 通用`: `图片`、`表格`、`文件`、`同步块`、`代码块`、`在线文档`、`模板内容`、`表情`、`标签`、`公式`
  - `样式布局`: `高亮块`、`分栏`、`分割线`、`素材库`
  - `数据`: `电子表格`
  - `画板图形`: `白板`、`脑图`、`路线图`、`流程图`、`文本绘图`
  - `团队协作`: `提及人`、`群聊`、`Agoal`、`钉钉日程`、`视频会议`、`知识库`、`信息收集`、`投票`
  - `项目管理`: `项目任务`、`项目看板`
  - `小工具`: `目录`、`表态`、`日期`、`投屏`、`计时器`、`倒计时`、`手写签名`、`模板按钮`、`内嵌网页`、`三方应用`、`企业内应用`
- `正文` 样式菜单包含：`正文`、`标题1`、`标题2`、`标题3`、`标题4`、`标题5`、`标题6`。
- `默认` 字体菜单包含：`默认`、`宋体`、`微软雅黑`、`华文宋体`、`黑体`、`Arial`、`钉钉进步体`、`阿里巴巴普惠体2.0`、`阿里巴巴普惠体3.0` 等字体。
- `菜单` 入口展开后包含：`格式`、`布局`、`视图`、`工具`。
- `添加图标` 会打开图标面板；已验证可见项包括 `推荐`、`表情符号`、`图标`、`移除`、`全部`。添加后入口会变为 `隐藏图标`。
- `添加封面` 会打开封面面板；已验证可见项包括 `编辑封面`、`调整位置`、`官方图库`、`上传图片`、`移除`。添加后入口会变为 `隐藏封面`。
- `设置文档信息` 只验证到工具栏入口可见；未验证到可操作面板，不要写成确定流程。

## 定位插入点

优先让用户明确插入位置。如果用户没有指定，默认使用当前光标或文档末尾，但要在执行前说明假设。

常用定位方式：

1. 当前光标：直接插入。
2. 文档末尾：点击正文后按 `ControlOrMeta+End`，或滚动到底部后点击最后一个块之后。
3. 某段文字前后：用 `ControlOrMeta+F` 查找精确文本，点击目标段落后用方向键和 `Enter` 创建插入点。
4. 某个标题前后：查找标题文本，点击标题行，使用 `Home`、`End`、`ArrowUp`、`ArrowDown`、`Enter` 调整插入点。
5. 表格/代码块/图形块内部：先点击块内部，使块级编辑器或控制项出现，再执行对应操作。

每次定位后都要验证周边文本，确认插入点没有落在工具栏、标题栏或侧边导航中。

## 设置标题和正文样式

用于把段落改成正文或标题：

1. 选中目标段落，或把光标放到目标段落内。
2. 打开工具栏 `正文` 菜单。
3. 选择 `正文` 或 `标题1` 到 `标题6`。
4. 验证可见字号/层级变化，必要时读取 iframe 文本确认标题顺序。

不要把 `默认` 当成段落样式入口；当前验证中 `默认` 是字体菜单。

## 设置字体

用于修改选中文本或当前输入点字体：

1. 选中文本，或把光标放到目标段落内。
2. 打开工具栏 `默认` 菜单。
3. 选择可见字体，例如 `默认`、`宋体`、`微软雅黑`、`黑体`、`Arial`、`钉钉进步体`。
4. 截图或目视确认字体变化。

只有选择当前菜单中可见的字体。不同租户或版本的字体列表可能不同。

## 设置或移除文档图标

使用工具栏 `添加图标`：

1. 点击 `添加图标` 打开图标面板。
2. 在已验证的 `推荐`、`表情符号`、`图标` 或 `全部` 分类中选择图标。
3. 添加后验证工具栏入口变为 `隐藏图标`，并检查标题附近是否出现图标。
4. 需要移除时，重新打开图标面板并选择 `移除`；如果入口显示 `隐藏图标`，点击它可恢复到无图标状态。

不要把图标选择写死为某个具体图标，除非用户明确指定。

## 设置或移除文档封面

使用工具栏 `添加封面`：

1. 点击 `添加封面` 打开封面面板。
2. 只使用已验证的非 AI 路径：`官方图库` 或 `上传图片`。
3. 如果选择上传图片，上传前确认用户明确授权了具体本地文件。
4. 添加后验证工具栏入口变为 `隐藏封面`，并检查标题上方是否出现封面。
5. 需要调整时使用 `编辑封面` 或 `调整位置`。
6. 需要移除时使用 `移除`；如果入口显示 `隐藏封面`，点击它可恢复到无封面状态。

不要使用封面面板中的 AI 生成入口。

## 插入图片、文件、在线文档

使用 `插入` 菜单：

- 图片：`插入` -> `图片`
- 文件：`插入` -> `文件`
- 在线文档：`插入` -> `在线文档`

这些操作通常会打开上传、选择器或文档选择面板。上传本地文件、选择企业文档或发送外部数据前，必须确认用户明确授权了具体文件或目标数据。

如果 `插入` 菜单在当前浏览器运行时无法稳定展开，停止自动点击并改用斜杠命令或请用户手动打开菜单；不要根据历史菜单项盲选。

## 插入公式、标签、表情

使用 `插入` 菜单：

- 公式：`插入` -> `公式`
- 标签：`插入` -> `标签`
- 表情：`插入` -> `表情`

公式适合数学表达式；标签和表情适合短标记。插入后检查它们是否保持为原生块或 inline 元素，不要只看粘贴文本。

## 插入样式布局块

使用 `插入` 菜单：

- 高亮块：`插入` -> `高亮块`
- 分栏：`插入` -> `分栏`
- 分割线：`插入` -> `分割线`

插入高亮块后，把说明、提示、风险或结论放入块内。插入分栏后，分别点击每一栏填写内容。插入分割线后，确认它位于目标段落之间。

## 插入目录、日期和表态

使用 `插入` 菜单的小工具分组：

- 目录：`插入` -> `目录`
- 日期：`插入` -> `日期`
- 表态：`插入` -> `表态`

目录依赖文档标题层级；插入前先确保目标段落已用 `正文` 菜单设置为 `标题1` 到 `标题6`。日期适合计划、里程碑和会议记录。表态适合收集简单反馈。

## 使用菜单分类

`菜单` 入口已验证会展开 `格式`、`布局`、`视图`、`工具`。

使用方式：

- 需要批量格式、页面布局、视图切换或工具类能力时，先打开 `菜单`。
- 只根据打开后可见的二级菜单继续操作；不要猜测隐藏项。
- 如果菜单操作会修改整篇文档布局或视图，先说明影响范围。
