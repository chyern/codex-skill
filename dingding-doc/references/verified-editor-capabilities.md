# 已验证编辑能力

本文件是 agent 使用钉钉文档 Web 编辑器时的能力边界。只把这里或当前页面中实际可见、可验证的入口当作确定能力。

## 基础能力

- 可在 `alidocs.dingtalk.com` 文档页找到 `iframe#wiki-doc-iframe`。
- 可用 `frame.locator('body').innerText()` 读取 iframe 渲染文本。
- 可见文本曾包含：`菜单`、`插入`、`AI 创作`、`默认`、`正文`、`添加图标`、`添加封面`、`设置文档信息`。
- `AI 创作` 入口存在但禁用：不要点击，不要把它作为生成文档内容的路径。
- `插入` 菜单中也曾看到 `AI 听记`；它属于 AI/听记入口，同样禁用。
- 工具栏入口可见不等于可稳定点击；点击后必须用展开文本或截图验收。

## 内容生成和编辑

已验证可作为 agent 工作策略：

- 长篇内容由 agent 生成 Markdown/纯文本后粘贴。
- 修改已有内容时，使用 iframe 文本定位和截图辅助确认。
- 表格、代码块、流程图、文本绘图等原生块优先通过中文斜杠命令或已展开的 `插入` 菜单创建。
- 图片、文件、在线文档、同步块、模板内容、公式、日期、目录、分栏、高亮块、电子表格、任务、看板、投票、日程、会议、知识库、内嵌网页等组件已验证入口可见；具体创建面板按当前可见文本操作。
- 修改后必须读取 iframe 文本并检查可见布局。

## 段落、标题和字体

已验证：

- `正文` 样式菜单包含：`正文`、`标题1`、`标题2`、`标题3`、`标题4`、`标题5`、`标题6`。
- `默认` 字体菜单包含：`默认`、`宋体`、`微软雅黑`、`华文宋体`、`黑体`、`Arial`、`钉钉进步体`、`阿里巴巴普惠体2.0`、`阿里巴巴普惠体3.0` 等字体。

使用规则：

- 章节标题层级使用 `正文` 菜单，不使用 `默认` 菜单。
- 字体族使用 `默认` 菜单，只选择当前菜单中可见的字体。
- 格式修改后用截图确认字号、加粗程度或标题层级。

## 插入菜单能力

已验证 `插入` 菜单包含以下非 AI 项：

- `基础 / 通用`: `图片`、`表格`、`文件`、`同步块`、`代码块`、`在线文档`、`模板内容`、`表情`、`标签`、`公式`
- `样式布局`: `高亮块`、`分栏`、`分割线`、`素材库`
- `数据`: `电子表格`
- `画板图形`: `白板`、`脑图`、`路线图`、`流程图`、`文本绘图`
- `团队协作`: `提及人`、`群聊`、`Agoal`、`钉钉日程`、`视频会议`、`知识库`、`信息收集`、`投票`
- `项目管理`: `项目任务`、`项目看板`
- `小工具`: `目录`、`表态`、`日期`、`投屏`、`计时器`、`倒计时`、`手写签名`、`模板按钮`、`内嵌网页`、`三方应用`、`企业内应用`

另见 `AI 听记`，但不作为可用能力。

如果当前运行时无法稳定展开 `插入` 菜单：

- 不要根据历史菜单项盲选。
- 对可斜杠创建的块使用中文完整斜杠命令。
- 对上传、企业选择、权限或外部发送类操作，先要求用户明确授权并协助完成界面状态。

## 文档外观入口

已验证：

- `添加图标` 会打开图标面板；可见项包括 `推荐`、`表情符号`、`图标`、`移除`、`全部`。添加后入口可能变为 `隐藏图标`。
- `添加封面` 会打开封面面板；可见项包括 `编辑封面`、`调整位置`、`官方图库`、`上传图片`、`移除`。添加后入口可能变为 `隐藏封面`。
- `设置文档信息` 只验证到入口可见；未验证到稳定可操作面板，不要写成确定流程。

使用规则：

- 图标或封面是可见外观变更，只在用户明确要求时操作。
- 不要使用封面面板中的 AI 生成入口。
- 上传图片前确认具体本地文件和目标文档。

## 菜单分类

已验证 `菜单` 入口展开后包含：`格式`、`布局`、`视图`、`工具`。

使用规则：

- 只根据打开后可见的二级菜单继续。
- 页面布局、视图切换、整篇格式等可能影响整篇文档，执行前说明影响范围。
- 未验证到具体面板的功能，不要写成确定步骤。
