# 钉钉文档编辑器界面参考

本文件记录 agent 操作 `alidocs.dingtalk.com` Web 编辑器时可依赖的界面结构、入口文本和 fallback。

## 页面结构

- 文档正文通常位于 `iframe#wiki-doc-iframe`。
- iframe 内可读取工具栏和正文渲染文本。
- 外层页面可能只包含导航、分享入口和 iframe 壳，不要用外层正文判断文档内容。

读取示例：

```js
const frame = tab.playwright.frameLocator('#wiki-doc-iframe');
const text = await frame.locator('body').innerText({ timeoutMs: 5000 });
```

## 已见工具栏入口

iframe 工具栏可见入口包括：

- `菜单`
- `插入`
- `AI 创作`
- `默认`
- `正文`
- `添加图标`
- `添加封面`
- `设置文档信息`

只把 `AI 创作` 作为需要避开的入口，不要点击或使用。

工具栏入口可能只在渲染文本或截图中可见，不一定是稳定可点击的 role/button。点击后必须确认展开项出现；没有展开时停止猜坐标。

## 插入菜单项

打开 `插入` 后，已验证的非 AI 分组和项目：

- `基础 / 通用`: `图片`、`表格`、`文件`、`同步块`、`代码块`、`在线文档`、`模板内容`、`表情`、`标签`、`公式`
- `样式布局`: `高亮块`、`分栏`、`分割线`、`素材库`
- `数据`: `电子表格`
- `画板图形`: `白板`、`脑图`、`路线图`、`流程图`、`文本绘图`
- `团队协作`: `提及人`、`群聊`、`Agoal`、`钉钉日程`、`视频会议`、`知识库`、`信息收集`、`投票`
- `项目管理`: `项目任务`、`项目看板`
- `小工具`: `目录`、`表态`、`日期`、`投屏`、`计时器`、`倒计时`、`手写签名`、`模板按钮`、`内嵌网页`、`三方应用`、`企业内应用`

快捷提示可能冲突，例如 `图片`、`投票`、`投屏` 都可能显示 `/tp`。用中文完整斜杠命令，避免短别名。

## 斜杠命令优先项

菜单点击不稳定时，优先在正文插入点输入：

- `/表格`
- `/代码块`
- `/流程图`
- `/文本绘图`
- `/公式`
- `/高亮块`
- `/分栏`
- `/分割线`
- `/目录`
- `/日期`

选择命令前读取可见候选项，选择精确中文名称。

## 格式菜单

- `正文` 样式菜单：用于 `正文`、`标题1`、`标题2`、`标题3`、`标题4`、`标题5`、`标题6`。
- `默认` 字体菜单：用于字体族，例如 `默认`、`宋体`、`微软雅黑`、`黑体`、`Arial`。

不要把 `默认` 菜单当作标题层级入口。

## 图标、封面、文档信息

- `添加图标` 打开图标面板。添加后入口可能变为 `隐藏图标`。
- `添加封面` 打开封面面板。添加后入口可能变为 `隐藏封面`。
- `设置文档信息` 只验证到入口可见，未验证到稳定可操作面板。

图标、封面属于文档外观变更；用户未明确要求时不要修改。上传封面图片前必须确认具体本地文件。

## 菜单失败处理

满足任一条件时停止自动点击：

- 文本 locator 唯一但点击后没有展开项。
- 坐标点击落入正文或改变光标位置。
- DOM 快照不可用或可见 DOM 不暴露目标按钮。
- 截图显示菜单未打开。

下一步选择：

- 改用中文完整斜杠命令。
- 让用户手动打开目标菜单或把光标放到目标位置。
- 只给出操作方案，不修改在线文档。
